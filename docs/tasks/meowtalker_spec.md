# [TASK] Implémentation du MeowTalker 3000

## 1. Contexte & Discussion (Narratif)
> *L'utilisateur a exprimé le besoin de mieux communiquer avec ses félins, surtout lors des épisodes de zoomies nocturnes.*
- Discussion initiale sur l'incompréhension inter-espèces.
- Décision d'utiliser un modèle de langage spécialisé dans les onomatopées félines.

## 2. Fichiers Concernés
- `src/logic/meow_translator.js`
- `public/index.html`
- `src/styles/meow.css`

## 3. Objectifs (Definition of Done)
* Le système doit être capable de distinguer un miaulement d'un ronronnement.
* L'interface doit afficher la traduction avec une esthétique "premium" et des animations de moustaches.
* Une alerte sonore doit être déclenchée si le chat demande explicitement de renverser un verre d'eau.
